kfs-ui

自己学习vue3，写着玩的一个组件库.

介绍

该项目是自己学习vue3写着玩的，没啥大用。

技术

jsx + ts + vue3 + scss。

安装

npm i kfs-ui
yarn add kfs-ui
1
2

引用

import { createApp } from 'vue'
import App from './App.vue'
import KfsUI, { CcButton, CcLink } from 'kfs-ui/lib'
import 'kfs-ui/lib/style.css'

const app = createApp(App)

app
.use(KfsUI)
.use(CcButton)
.use(CcLink)
.mount('#app')

  text: 'text'
})

  text: 'text',
  success() {
    console.log('嗯。。。')
  }
})

  '--theme-**-**': '**'
